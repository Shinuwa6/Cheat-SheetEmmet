Introduction Emmet

Post by Nicolas Phimasane, Juil 8, 2016.
Introduction

Emmet est un système performant d'abréviations qui convertit quelques lettres en ligne CSS ou HTML complète.

Comment ça marche ?

Post by Nicolas Phimasane, Juil 8, 2016.
CçM ?

Il faut bien l'admettre, écrire en demande du temps, avec toutes ces balises, ces attributs, guillemets, parenthèses, etc. La plupart des éditeurs de texte proposent une complétion de code, qui est certes très utile, mais il reste encore beaucoup à taper. Emmet transforme de simples abréviations en code complexe.

Exemple d'utilisation

Post by Nicolas Phimasane, Juil 8, 2016.
Exemple

  • html:5 ou ! pour un doctype HTML5
  • html:xt pour un doctype transitionnel XHTML
  • html:4s pour un doctype strict HTML

Il existe des tas d'autres raccourcis EMMET, je vous conseil de consulter la doc officiel.
